as part of backend development course, I built a Travel Booking API.
You can see the github [**repository**](https://github.com/iLiranS/natours-api). 
You can also see full lis of endpoints in the [API docs](https://lirans.vercel.app/blog/07-natours/natours-docs.html).

## Features
- User authentication (register, login, logout , forgot password, reset password)
- Booking management (create, update, delete)
- Tour management (create, update, delete)
- Tour booking (create, update, delete)
- Tour search (search by name, price, duration)
- Tour details (get tour details by id)
- Tour reviews (create, update, delete)
- Tour ratings (create, update, delete)
- Tour images (upload, delete)
- Tour categories (create, update, delete)
- Tour tags (create, update, delete)
- Tour tags (create, update, delete)
