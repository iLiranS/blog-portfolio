A Multiplayer card game made for React Jam using [Rune SDK](https://www.rune.ai/) platform
You can play it [locally](https://app.rune.ai/dev-8rORCUhb) or using their application.

## Introduction
inspired by liars bar game, I've created a Multiplayer mobile game using React.
The game was made in 4 days total as part of the React JAM challenge and got nominated (4th place in total playtime).

## Tech Stack
Rune-SDK , React, Zustand, Tailwind, Typescript
